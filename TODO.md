- [ ] **고급 템플릿 기능**
  - 템플릿 폴더에 한글 파일과 동일한 이름의 YAML 파일이 있으면 고급 템플릿으로 인식
    - 예: `보고서.hwp` + `보고서.yaml`
    - 예: `보고서.hwpx` + `보고서.yaml`
  - 한글 파일 본문 안의 `{{변수명}}` 패턴을 템플릿 변수로 인식
  - YAML 파일에서 각 변수의 입력 방식과 기본값을 정의
  - 변수 입력 방식 지원
    - `select`: YAML에 정의된 리스트 값 중 하나를 사용자가 선택
    - `text`: 사용자가 직접 입력
    - `date`: 현재 날짜, 지정 포맷 등 날짜 값을 자동 생성
    - `script`: 현재 날짜/작성자/프로젝트 변수 등 계산 가능한 값을 스크립트 또는 제한된 표현식으로 생성
  - 템플릿 선택 시 YAML을 읽어 변수 입력 UI를 자동 생성
  - 사용자가 선택/입력한 값을 문서의 `{{변수명}}` 위치에 치환
  - 프로젝트 기본 변수와 템플릿 YAML 변수의 우선순위 규칙 정의
  - 스크립트 실행 범위와 보안 정책 결정 (브라우저 샌드박스, 허용 함수 제한 등)
  - AI에게 고급 템플릿 변수 목록과 선택 가능한 값을 컨텍스트로 전달
