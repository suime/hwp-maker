# hwp-maker 디자인 명세 (DESIGN.md)

> 이 문서는 hwp-maker의 UI/UX 레이아웃을 정의합니다.
> 실제 구현 전 이 문서를 기준으로 컴포넌트를 설계하세요.

---

## 1. 전체 레이아웃 구조

```
┌─────────────────────────────────────────────────────────────────┐
│                        TopBar (헤더)                             │
├────┬──────────┬──┬──────────────────────────────────────────────┤
│    │          │  │                                              │
│아이│  패널 내용│  │              메인 영역                        │
│콘  │  (탭 내용)│  │           MainPanel                          │
│레일│           │드│       (hwpx 편집 + 미리보기)                  │
│    │          │래│                                              │
│    │          │그│                                              │
└────┴──────────┴──┴──────────────────────────────────────────────┘
  48px  가변(드래그)  4px(핸들)       나머지 전체
```

### 비율 및 크기

| 영역 | 너비 | 비고 |
|---|---|---|
| 아이콘 레일 (IconRail) | `48px` (고정) | 탭 전환 아이콘 세로 나열 |
| 사이드바 패널 (SidePanel) | `280px` (기본, 드래그 조절) | 최소 `180px`, 최대 `520px` |
| 리사이즈 핸들 | `4px` | 드래그로 사이드바 너비 조절 |
| MainPanel | 나머지 전체 | flex-grow |
| TopBar | 전체 너비 | `48px` 높이 |

## 2. TopBar (상단 헤더)

```
┌─────────────────────────────────────────────────────────────────┐
│ [☰] hwp-maker          [파일명.hwpx ▾]   [저장] [↓ 내보내기]   │
└─────────────────────────────────────────────────────────────────┘
```

### 구성 요소

| 요소 | 위치 | 설명 |
|---|---|---|
| 사이드바 토글 버튼 `[☰]` | 좌측 | 사이드바 접기/펼치기 |
| 로고 + 앱명 | 좌측 | `hwp-maker` 브랜드 |
| 현재 파일명 | 중앙 | 편집 중인 문서명, 클릭 시 이름 변경 |
| 저장 버튼 | 우측 | Session Storage에 현재 상태 저장 |
| 내보내기 버튼 | 우측 | hwp / hwpx 포맷으로 로컬 다운로드 |

## 3. 사이드바 영역

### 3-1. 아이콘 레일 (IconRail)

```
┌────┐
│ ☰  │  ← 사이드바 토글 (펼침/접힘)
├────┤
│ 💬 │  ← 채팅 탭
│ 📄 │  ← 템플릿 탭
│ ⚙  │  ← 설정 탭
│    │
│    │
│    │
│    │
└────┘
```

- 너비 `48px` 고정, 항상 표시
- 선택된 탭 아이콘에 브랜드 컬러 강조
- 사이드바 토글 버튼으로 패널 전체 접기/펼치기 가능

### 3-2. 사이드바 패널 (SidePanel)

```
┌──────────────────────┐
│ [탭 제목]            │  ← 패널 헤더
├──────────────────────┤
│                      │
│  탭 내용             │  ← 선택된 탭에 따라 교체
│                      │
└──────────────────────┘
```

### 3-2. 사이드바 탭 목록

| 탭 | 아이콘 | 설명 |
|---|---|---|
| **채팅** | 💬 | AI 어시스턴트 채팅 화면 |
| **템플릿** | 📄 | 미리 정의된 hwp 양식 목록 및 관리 |
| **설정** | ⚙ | AI API 설정, 앱 설정 |

### 3-3. 리사이즈 핸들

- 패널 우측에 `4px` 드래그 핸들 배치
- 마우스 드래그로 SidePanel 너비 조절 (`min: 180px`, `max: 520px`)
- 조절된 너비는 `localStorage`에 저장하여 새로고침 후에도 유지
- 핸들 호버 시 커서 `col-resize` 로 변경, 브랜드 컬러 강조선 표시

## 4. 사이드바 탭 상세

### 4-1. 채팅 탭 (💬)

```
┌──────────────────────┐
│ AI 어시스턴트         │  ← 탭 헤더
├──────────────────────┤
│                      │
│  [AI] 안녕하세요!     │
│                      │
│        [사용자] 입력  │
│                      │
│  [AI] 처리 중...     │
│                      │
│                      │
├──────────────────────┤
│ [입력창          ] [▶]│  ← 하단 고정 입력창
└──────────────────────┘
```

**세부 명세:**
- AI 메시지: 좌측 정렬, 배경색 패널
- 사용자 메시지: 우측 정렬, 브랜드 컬러 배경
- 입력창: `Shift+Enter` 줄바꿈, `Enter` 전송
- 스트리밍 응답 지원 (타이핑 효과)
- 로딩 중 점 애니메이션 표시

---

### 4-2. 템플릿 탭 (📄)

```
┌──────────────────────┐
│ 템플릿               │  ← 탭 헤더
│             [+ 추가] │
├──────────────────────┤
│ 🔍 검색...           │
├──────────────────────┤
│ ▼ 기본 템플릿             │
│   📝 업무계획(방침)       │
│   📝 1P보고서             │
├──────────────────────┤
│ ▼ 내 템플릿          │
│   📝 (없음)          │
│                      │
└──────────────────────┘
```

**기본 내장 템플릿:**

| 템플릿명 | 설명 |
|---|---|
| 업무계획(방침) | 부서/팀 업무 계획 및 방침 작성용 양식 |
| 1P보고서 | 핵심 내용을 1페이지로 요약하는 보고서 양식 |

**세부 명세:**
- 템플릿 클릭 시 메인 패널에 해당 양식 로드
- `+ 추가`: 로컬 hwp/hwpx 파일을 업로드하여 템플릿으로 등록
- 내 템플릿: 사용자가 등록한 커스텀 양식 (`localStorage` 저장)
- 템플릿 항목 우측 `[×]` 버튼으로 삭제 (기본 템플릿은 삭제 불가)

---

### 4-3. 설정 탭 (⚙)

```
┌──────────────────────┐
│ 설정                  │  ← 탭 헤더
├──────────────────────┤
│ ▼ AI 설정            │
│  Base URL  [______]  │
│  API Key   [______]  │
│  모델명    [______]  │
│            [연결 테스트]│
├──────────────────────┤
│ ▼ 편집기 설정         │
│  미리보기 크기  [___] │
│  자동 저장   [ON/OFF] │
├──────────────────────┤
│            [저장]     │
└──────────────────────┘
```

**세부 명세:**
- AI 설정은 `localStorage`에 저장
- 연결 테스트: 현재 설정으로 간단한 ping 요청 후 성공/실패 표시
- 편집기 설정: 미리보기 용지 크기, 자동 저장 주기 등

## 5. MainPanel (메인 영역)

### 5-1. 메인 패널 구조

```
┌──────────────────────────────────────────────┐
│  [편집] [미리보기]                  [용지 비율▾] │  ← 뷰 토글 탭
├──────────────────────────────────────────────┤
│                                              │
│   ┌──────────────────────────────────────┐  │
│   │                                      │  │
│   │          hwp 문서 콘텐츠              │  │
│   │       (rhwp WASM 렌더링 영역)         │  │
│   │                                      │  │
│   └──────────────────────────────────────┘  │
│                                              │
└──────────────────────────────────────────────┘
```

### 5-2. 뷰 모드

| 모드 | 설명 |
|---|---|
| **편집** | rhwp WASM 기반 직접 편집 (커서, 타이핑 가능) |
| **미리보기** | 읽기 전용 렌더링 (인쇄 레이아웃 그대로) |

### 5-3. 문서 표시 영역

- 배경: 회색 (용지 외곽 구분)
- 용지: A4 기준 흰색 박스, 그림자 효과
- 스크롤: 세로 스크롤로 여러 페이지 탐색
- 여백: 상하좌우 한글 기본 여백 준수

## 6. 반응형 처리 (미래 고려사항)

| 화면 너비 | 처리 방식 |
|---|---|
| `≥ 1280px` | 사이드바 기본 펼침 |
| `768px ~ 1279px` | 사이드바 아이콘 모드 (48px) |
| `< 768px` | 사이드바 오버레이 모드 (슬라이드) |

